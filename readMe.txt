主目录：北邮校园：用于求解北邮校园网络的相关代码，网络拓扑图见报告的5.4.2
	     随机网络：是生成并求解一个随机网络的VRP问题的代码合集，该部分为了方便调用，因此写成了许多类和一个main程序，需要注意的是车辆最大装载量的定义位于class-Rnetwork的第10行，与其相邻的是生成随机数所选择的种子。本算法也支持经典算例如Set系列或是Golden系列的求解，仅需修改main函数中导入网络的部分即可（保证距离矩阵对应）
	      GIS处理：是使用QGIS选定研究区域、兴趣点后进行初步处理的qgz文件，该文件主要包含了一个OSM底图，一个兴趣点的图层和一个研究区域道路网络的图层。


子目录：北邮校园：class-distance是读取network.csv的网络并且计算曼哈顿距离的程序，class-gene和class-saving分别是用遗传算法和节约里程法求解的程序
	     随机网络：class-ALNS是二阶段自适应大邻域搜索的代码，除此之外class-saving是节约里程法，class-gene是遗传算法，遗传算法的内部未用类编写，因为这是个最简单的遗传算法，在这个大随机网络下表现极差，没有单独在主程序中调用研究的必要。其余文件均按照类的形式编写，方便调用，对于关键参数也写了get方法。class-Rnetwork是生成随机网络相关的文件，调用旋转射线法分区的代码也写在了这个文件里，之所以这样写是因为同一个文件同一个类里面不用考虑成员的访问权限问题。调用这个文件生成的网络和距离矩阵将放置在result文件夹里，之后的算法求解也是调用这些文件（data文件夹是留给以后研究具体网络的，目前data文件夹没用，里面是测试用的一个50节点随机网络），所有的结果最后是放在了result的partitions文件夹里。
